# Junior Security Analyst Introduction

## Overview

This room introduces the role of a Junior Security Analyst within a Security Operations Center. It outlines key responsibilities, daily tasks, and expectations for entry-level SOC analysts.

**Path:** SOC Level 1

**Difficulty:** Easy

**Focus Area:** SOC Operations, Analyst Responsibilities



## What Was Covered

* Role of a Junior Security Analyst
* SOC responsibilities and daily activities
* Alert monitoring and triage basics
* Escalation and communication processes



## Key Points

* Junior analysts are the first line of defense
* Alert triage and investigation are core duties
* Clear documentation and escalation are critical
* Team collaboration is essential in a SOC
