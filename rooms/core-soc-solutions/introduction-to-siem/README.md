# Introduction to SIEM

## Overview
This sub-room introduces Security Information and Event Management (SIEM) and its role in Security Operations Center (SOC) environments. It explains how SIEM platforms collect, correlate, and analyze security events to support threat detection and investigation.

**Path:** SOC Level 1  

**Focus Area:** SIEM, Log Analysis, Security Monitoring



## What Was Covered
- Purpose of SIEM in a SOC
- Event and log aggregation
- Correlation and alert generation
- Analyst interaction with SIEM platforms



## Key Concepts
- Centralized log collection
- Event correlation and normalization
- Alerting on suspicious activity
- Investigation using search and filters



## Key Points
- SIEM provides centralized security visibility
- Correlation improves detection accuracy
- Analysts rely on SIEM for alert triage
- Quality logging directly impacts detection
