# Introduction to EDR

## Overview
This sub-room introduces Endpoint Detection and Response (EDR) and its role in modern SOC environments. It explains how EDR tools provide visibility into endpoint activity and support threat detection and incident response.

**Path:** SOC Level 1  
 
**Focus Area:** Endpoint Security, Detection and Response



## What Was Covered
- Purpose of EDR in a SOC
- Differences between EDR and traditional antivirus
- Endpoint visibility and telemetry
- Detection and response capabilities



## Key Concepts
- Continuous monitoring of endpoint activity
- Behavioral-based threat detection
- Investigation of suspicious processes
- Containment and remediation actions



## Key Points
- EDR provides deep visibility into endpoints
- Detection is based on behavior, not signatures alone
- SOC analysts use EDR for investigation and response
- EDR complements SIEM and network-based monitoring
