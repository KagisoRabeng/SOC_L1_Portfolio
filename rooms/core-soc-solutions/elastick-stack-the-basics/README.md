# Elastic Stack: The Basics

## Overview
This sub-room introduces the Elastic Stack and its role in modern SOC environments. It explains how Elastic components work together to collect, store, search, and visualize security data for monitoring and investigation.

**Path:** SOC Level 1
 
**Focus Area:** SIEM, Log Analysis, Security Monitoring



## What Was Covered
- Purpose of the Elastic Stack in a SOC
- Overview of Elastic Stack components
- Log ingestion and data indexing
- Searching and visualizing security events



## Elastic Stack Components
- **Elasticsearch** – Stores and indexes data for fast searching
- **Logstash** – Processes and ingests logs from multiple sources
- **Beats** – Lightweight agents that ship data to Elasticsearch
- **Kibana** – Visualizes data and supports investigation workflows



## Key Points
- Elastic provides centralized visibility across environments
- Fast search enables efficient alert investigation
- Visual dashboards support threat detection
- Elastic is commonly used as a SIEM platform
