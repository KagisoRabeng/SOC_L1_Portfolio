# Splunk: The Basics

## Overview
This sub-room introduces Splunk and its role as a SIEM platform in Security Operations Center (SOC) environments. It focuses on how Splunk is used to collect, search, and analyze machine data for security monitoring and investigations.

**Path:** SOC Level 1  
 
**Focus Area:** SIEM, Log Analysis, Security Monitoring



## What Was Covered
- Purpose of Splunk in a SOC
- Overview of Splunk components
- Indexing and searching data
- Basic investigation workflows



## Key Concepts
- Data ingestion and indexing
- Searching with Splunk Processing Language (SPL)
- Event analysis and filtering
- Dashboards and visualizations



## Key Points
- Splunk provides centralized log visibility
- SPL enables powerful search and analysis
- SOC analysts use Splunk for alert investigation
- Data quality affects detection effectiveness
