# Phishing Emails in Action

## Overview
This sub-room focuses on analyzing real-world phishing email examples. It helps SOC analysts apply phishing concepts by identifying malicious indicators within email content and understanding attacker techniques.

**Path:** SOC Level 1  
  
**Focus Area:** Phishing Analysis, Email Investigation



## What Was Covered
- Identifying phishing indicators in emails
- Analyzing sender details and email content
- Spotting malicious links and attachments
- Understanding attacker tactics used in phishing



## Key Indicators Analyzed
- Suspicious sender addresses
- Urgent or manipulative language
- Malicious or misleading URLs
- Unexpected attachments



## Key Points
- Phishing emails often mimic trusted entities
- Content analysis is critical for detection
- Multiple indicators should be correlated
- User reports help initiate investigations
