# Phishing Analysis Tools

## Overview
This sub-room introduces common tools used by SOC analysts to analyze phishing emails. It focuses on how these tools help identify malicious indicators without interacting with potentially harmful content.

**Path:** SOC Level 1  

**Focus Area:** Phishing Analysis, Threat Investigation

---

## What Was Covered
- Purpose of phishing analysis tools
- Safe analysis of suspicious emails
- Extracting and analyzing indicators
- Supporting SOC investigation workflows

---

## Common Analysis Tools
- URL and domain reputation checkers
- File hash analysis tools
- Email header analysis tools
- Sandbox and detonation platforms

---

## Key Points
- Tools reduce risk during analysis
- Indicators help confirm malicious intent
- Multiple tools should be used for validation
- Analysis results support escalation decisions
