# Snapped Phish-ing Line

## Overview
This sub-room focuses on analyzing phishing attempts using a structured, hands-on approach. It reinforces how SOC analysts identify, extract, and assess phishing indicators from real-world examples.

**Path:** SOC Level 1  

**Focus Area:** Phishing Analysis, Indicator Identification



## What Was Covered
- Reviewing phishing email content
- Identifying malicious indicators
- Analyzing URLs, domains, and sender details
- Applying analysis steps used in SOC workflows



## Indicators Identified
- Suspicious sender information
- Malicious or misleading URLs
- Social engineering language
- Indicators of compromise (IOCs)



## Key Points
- Phishing analysis requires attention to detail
- Multiple indicators should be validated together
- Structured analysis improves accuracy
- Findings support escalation and response decisions
