# SOC Metrics and Objectives

## Overview
This sub-room introduces key metrics and objectives used to measure the effectiveness of a Security Operations Center (SOC). It explains how metrics help SOC teams evaluate performance, improve detection, and support management decisions.

**Path:** SOC Level 1  
**Difficulty:** Easy  
**Focus Area:** SOC Operations, Metrics and Performance

---

## What Was Covered
- Purpose of SOC metrics
- Common SOC performance metrics
- Aligning metrics with SOC objectives
- Using metrics for continuous improvement

---

## Common SOC Metrics
- Mean Time to Detect (MTTD)
- Mean Time to Respond (MTTR)
- Alert volume and trends
- False positive rates
- Incident escalation rates

---

## SOC Objectives
- Detect threats as early as possible
- Reduce response and containment time
- Minimize business impact
- Continuously improve security posture

---

## Key Points
- Metrics provide visibility into SOC performance
- Objectives guide SOC priorities and actions
- Accurate data supports informed decision-making
- Metrics should align with business goals
