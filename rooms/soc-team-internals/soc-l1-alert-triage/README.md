# SOC L1 Alert Triage

## Overview
This sub-room focuses on the alert triage process performed by SOC Level 1 analysts. It explains how analysts review, validate, prioritize, and escalate security alerts generated by monitoring tools.

**Path:** SOC Level 1  
  
**Focus Area:** SOC Operations, Alert Triage



## What Was Covered
- Purpose of alert triage
- Understanding alert context
- Differentiating true positives from false positives
- Escalation and documentation procedures



## Alert Triage Process
- Review alert details and metadata
- Validate activity using logs and tools
- Assess severity and potential impact
- Escalate confirmed incidents or close false positives



## Key Points
- Not all alerts indicate real threats
- Context is critical during analysis
- Accurate triage reduces analyst fatigue
- Proper escalation supports faster response
